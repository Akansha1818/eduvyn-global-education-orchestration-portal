## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Akansha1818/eduvyn-global-education-orchestration-portal.git
   cd eduvyn-global-education-orchestration-portal
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment variables**
   - Copy `.env.example` to `.env` and update with your settings.

4. **Run the application**
   ```bash
   npm start
   ```
